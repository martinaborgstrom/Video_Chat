### Required
Installare:
- Node.js
- npm
- ReactJS

### How to start Web Application
Impostare le variabili d'ambiente:
```
PORT = <port_number>
JWT_SECRET = <jwt_secret>
DB_URI= <db_uri>
```

Installare le dipendenze e avviare server:
```
npm install
cd backend/src
node index.js
```
Avviare il frontend:
```
cd fronted
npm start
```

Accedere alla web application all'indirizzo ```http://localhost:3000```
